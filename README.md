# OneClickShop

Simple one click shhoping app using custome APIs backed by Retrofit.


### Features
- Show all the products 
- Search product by it's name
- User can add new product with image

### Topic covered
- Kotlin (Primary Language)
- Jetpack Compose
- Retrofit
- MVVM
